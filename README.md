# Healthcare Readmission Risk Prediction System

This project predicts patient readmission risk using a lightweight logistic regression model deployed on AWS Lambda with an interactive Streamlit dashboard.

## Features
- Logistic Regression ML model
- Serverless AWS (Lambda + API Gateway + S3)
- Free-tier optimized
- Streamlit frontend
- Flowchart + Technical Summary PDF included

## Project Structure
- `/lambda` — AWS Lambda code
- `/model` — Model parameters
- `/docs` — PDF documentation
- `streamlit_app.py` — Frontend UI
